# Hora CDMX Skill

Skill para consultar la hora actual en Ciudad de México.

## Uso

```bash
# Consultar hora
./hora-cdmx.sh

# Establecer hora como variable de entorno
./set-hora-cdmx.sh
```

## Configuración

- Zona horaria: America/Mexico_City (GMT-6)
- Formato: 12h (AM/PM)
- Idioma: Español

## Archivos

- `hora-cdmx.sh` - Muestra la hora actual
- `set-hora-cdmx.sh` - Establece TZ para sesión
