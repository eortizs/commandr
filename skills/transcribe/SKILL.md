# Transcribe Skill

Skill para transcribir audio a texto usando Groq Whisper.

## Uso

```bash
./transcribe.sh archivo.ogg
```

## Requisitos

- Groq API key configurada
- Archivos de audio en formato soportado (ogg, mp3, wav)

## Archivos

- `transcribe.sh` - Transcripción básica
- `transcribe-verbose.sh` - Transcripción con metadata
